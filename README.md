# 🔧 Can Tesisat - Corporate Business Website

A modern, fully responsive, and fast-loading single-page corporate website built for a local plumbing and natural gas business in Diyarbakır.

This project demonstrates my ability to deliver end-to-end freelance web solutions for real-world clients, focusing on user experience, mobile accessibility, and business conversion.

🔗 **Live Website:** https://cantesisat21.com

## 🚀 Technologies & Tools Used
* **Frontend:** HTML5, Vanilla JavaScript
* **Styling:** Tailwind CSS
* **Icons:** FontAwesome 6
* **Hosting & Deployment:** Güzel Hosting
* **Version Control:** Git & GitHub

## 💡 Key Features
* **Mobile-First Design:** Fully optimized with Tailwind CSS for seamless viewing on all mobile devices (crucial for emergency plumbing services).
* **Direct Communication:** Integrated click-to-call buttons and direct WhatsApp messaging links for immediate customer contact.
* **Smooth Navigation:** Implemented smooth scrolling and a custom JavaScript mobile menu for a seamless user experience.
* **Fast Performance:** Optimized architecture using CDN links for rapid page loads.
